## 📖 Overview

**Big Hearts** is a premium relief distribution platform designed to bridge the gap between global donors and local needs. Built with a "Minimalist-Premium" aesthetic, this project demonstrates a modern approach to humanitarian aid through technology.

The development focused on creating a scalable architecture using **Redux Toolkit** and **TypeScript** to ensure enterprise-grade type safety and predictable state management. By leveraging the MERN stack, the application maintains high performance while handling complex data flows across relief categories and donation tracking.

---

## ✨ Features

- **⚡ State-Managed Logic**: predictable and centralized data flow using **Redux Toolkit**.
- **🖼️ Cinematic Testimonials**: A single-slide, focused review experience using **Swiper 11** with `EffectFade`.
- **📊 Real-Time Analytics**: Visualizing relief distribution progress via **Recharts**.
- **🧩 Premium Animations**: High-end micro-interactions and scroll reveals powered by **Framer Motion**.
- **🛡️ Type-Safe Codebase**: 100% **TypeScript** coverage for enterprise-grade stability and developer experience.
- **🔔 Interactive Feedback**: Sleek user notifications provided by **Sonner**.

## 🚀 Tech Stack

### Frontend

- **Framework**: React 18 (Vite)
- **State Management**: Redux Toolkit (@reduxjs/toolkit)
- **Language**: TypeScript 5
- **Styling**: Tailwind CSS & DaisyUI
- **Icons**: Lucide React
- **Charts**: Recharts

### Backend (MERN)

- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: MongoDB
